## Android-SDK

Android-SDK是紫荆云平台Android端SDK。Android-SDK可以帮助您快速搭建基于Android平台的音视频应用，或者帮助您的应用具备音视频通讯的能力。Android-SDK使用WebRTC协议构建音视频通话服务，让您轻松应对一对一、多对多视频通话及实时数据传输等复杂应用场景。
